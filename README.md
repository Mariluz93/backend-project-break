# Tienda de ropa

Hemos creado una aplicación web que simula una tienda de ropa con un catálogo público de productos y un dashboard para el administrador. Esta aplicación ha sido desarrollada con Node.js y MongoDB. Además,  hemos subido las imágenes con Cloudinary. Incluye un filtrado por productos tanto en la web del cliente como en el de adiministrador. El panel de administrador está protegido por autentificación. Además, hemos incluido una API REST en formato JSON para permitir la integración futura con un frontend en React.

# Tecnologías utilizadas

- Node.js

- Express

- MongoDB Atlas

- Mongoose

- Cloudinary

- Express-session

- Method-override

- Render

- Template literals (SSR)
